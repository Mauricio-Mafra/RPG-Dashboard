# Projeto Módulo 3 - RPG Dashboard

Um dashboard para gerenciar partidas de RPG, oferecendo ferramentas para simplificar o andamento das campanhas como uma visualização simples das fichas de personagens do grupo, bem como a atualização de dados das mesmas.


# Dependências

O seguinte projeto utiliza as seguintes dependências para sua execução:

	

 - SadConsole
 - NAudio
